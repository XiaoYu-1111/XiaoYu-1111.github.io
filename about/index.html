关于我

👨‍🎓 个人简介

你好，我是 XIAO YU，毕业于 长安大学 (CHD)。

主要研究领域为 地震勘探波场模拟与数据分析处理。我致力于将 高性能计算 (HPC) 技术应用于地球物理领域，探索 GPU 并行计算在地震波场模拟（正反演）与实时监测系统中的工程化落地。

🔬 研究方向

  • 🌊 地震波场模拟: 基于波动方程的数值模拟（FD/FEM），复杂介质下的波场传播特性研究。
  • 🔍 浅地表地震探测: 基于绕射波的地震探测技术，包括浅层异常体探测、目标识别、绕射波信号处理等。
  • 🚀 高性能计算: 基于 CUDA/OpenCL 的 GPU 并行加速技术，大规模地震数据处理优化。
  • 🖥️ 实时监测系统: 结合 ImGui 与 C++ 开发轻量级、可视化的实时地震台站监测软件 (SeisMonitor)。

🚀 项目经历 (Projects)

  • 基于 GPU 的波场模拟加速器

    利用 Compute Shader 技术优化传统有限差分算法，大幅提升正演模拟效率，实现针对大规模网格的高速计算。

  • Seis_Simulation 地震波场模拟

    基于 C++ 与 Qt、OpenMP 开发的地震波场有限差分求解器。

    • 支持多种波源（雷克子波等）、复杂介质模型导入。
    • 实现了 PML 吸收边界条件与自由表面边界条件的可视化配置。

  • SeisMonitor 实时地震台站监测系统

    基于 C++ 与 ImGui (即时模式 GUI) 开发的轻量级监测终端。

    • 实现波形数据的实时可视化绘制（高帧率）。
    • 集成DSP算法异常信号检测与自动预警功能。
